feat: self-evolving skill system - 3 codegen learning modules

Module 1 - CodeGenLibrary: auto-saves successful codegen for reuse.
  Keyword-based matching finds previous solutions, skipping LLM generation.
  LRU cache (200 entries), tracks usage count and execution time.

Module 2 - DynamicSkillRegistry: promotes codegen to named runtime skills.
  User can save_as_skill to register code as a callable tool.
  Persists to JSON, auto-loads on startup, appears in LLM tool list.

Module 3 - CodePatternLearning: tracks error patterns and API usage.
  Auto-detects recurring compile errors, records learned fixes.
  Injects warnings and usage stats into LLM context.

Integration:
- DynamicCodeSkill now uses all 3 modules
- AgentOrchestrator injects codegen context (library summary, dynamic skills, patterns)
- SkillRegistry supports RegisterDynamic for runtime skill addition
- WebViewBridge initializes and persists all codegen modules
- ChatSessionV2 passes modules through to AgentOrchestrator
