## Kích thước & Đơn vị

| Thuật ngữ | Ý nghĩa | Đơn vị |
|---|---|---|
| Đường kính (DN) | Nominal Diameter | mm (DN15, DN20, DN25...) |
| Kích thước ống gió | Duct Size (W x H) | mm (400x200, 600x300...) |
| Đường kính ống gió tròn | Round Duct Diameter | mm |
| Lưu lượng gió | Airflow | L/s, m³/h, CFM |
| Lưu lượng nước | Water Flow | L/s, m³/h, GPM |
| Vận tốc | Velocity | m/s, FPM |
| Áp suất | Pressure | Pa, mmWG, inWG |
| Tổn thất áp suất | Pressure Drop/Loss | Pa/m |
| Độ dốc | Slope | %, mm/m |
| Công suất lạnh | Cooling Capacity | kW, BTU/h, RT |
| Công suất nhiệt | Heating Capacity | kW, BTU/h |
| Nhiệt độ | Temperature | °C, °F |
